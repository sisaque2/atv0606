RF01 - Exibir uma fila de filmes e horários de exibição em cada cinema

RF02 - O sistema deve conter uma interface separada para as franquias especificarem suas sessões

RF03 - O sistema deve permitir o registro diário da quantidade de espectadores de cada sessão

RF04 - O sistema deve gerar a totalização e relatórios de público filtrados por sessão, por filme e por unidade do cinema

RF05 - O sistema deve permitir a consulta e o cadastro de informações detalhadas dos filmes, incluindo título, gêneros, diretores e elenco principal
